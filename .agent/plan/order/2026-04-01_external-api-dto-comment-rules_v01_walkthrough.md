# 外部接口 DTO 注释规则沉淀总结

## 本次变更

已在 `.agent/rules/AI_BEHAVIOR_RULES.md` 追加仓库级规则：

- 外部接口 DTO 需要按协议场景或版本分组字段
- DTO 字段和嵌套结构需要补充可脱离文档阅读的中文注释
- 注释需说明字段语义、适用场景和关键协议约束
- 已有协议说明、业务注释和 TODO 默认保留，不得擅自删除

## 适用范围

该规则适用于所有第三方接口接入代码，包括但不限于：

- SPI 回调 DTO
- OpenAPI 请求/响应 DTO
- 第三方 webhook / notify DTO
- 需要长期维护的协议适配层对象

## 结果

后续 AI 在处理外部接口 DTO 时，不应只做“字段能用就行”的最小实现，而应同步保证结构可读性和后续维护性。
