## 预期结论
- FastDFS 默认不能像本地文件系统那样由业务直接指定任意目录路径。
- 业务侧通常只能选择 `group`、上传方式及文件扩展名/元数据；真正的存储路径和远程文件名由 FastDFS 生成。
- 当前仓库上传实现走 `autoUpload(...)`，未暴露自定义路径能力。
