### 订单退款

**更新时间：** 2026-03-19

**无需业务授权**

---

#### 消息场景说明

`meituan.trip.order.refund` (订单退款)

##### 使用场景

美团向合作方发起退款申请，用于正常流程的用户退款，非客服强制退款

##### 接口说明

1.  美团向合作方提交退款申请中，退款流水号 `refundId` 是退款申请的唯一标识，请按照退款流水号进行拒绝、同意，切勿仅按照订单下票的份数。
2.  美团会用同一流水号去查退款进度，请不要因为已经审批中而返回退款失败，要继续返回 602 退款审批中。
3.  如果此退款流水号已经退款成功, 请返回退款成功（code 200）或已退款（code 601）。
4.  退款失败的 `describe` 要返回了真实的失败原因，不能用“其他”代替。`code` 按照美团给出的对接。
5.  超过 1 个工作日未处理的退款中订单，美团将默认商家同意退款以保证用户体验。

---

#### 公共参数

*   **HTTP URL：** `https://baseUrl`（供应商提供）（仅支持 https 协议且端口为 443）
*   **HTTP Param：** param 遵循 JSON 格式，ContentType 为 `application/x-www-form-urlencoded`, charset=utf-8
*   **HTTP 方法：** POST
*   **消息推送公共参数：** 消息推送公共参数
*   **消息响应公共规范：** 消息响应公共规范

#### 请求示例

> **注意：** 回调消息统一通过 HTTPS 请求推送，Post 方法，ContentType 为 `application/x-www-form-urlencoded`。开发者需要按照规范实现 HTTPS 接口，以接收推送的消息。

**非实名制订单退款请求**
```plain text
{
    "msgType": "6610031",
    "opBizCode": "1711590987",
    "developerId": "123456",
    "businessId": "66",
    "sign": "4656285a4c2493e279d929b8b9f4e29310datest",
    "msgId": "3702923312382104528",
    "timestamp": "1711592316026",
    "message": "{\"partnerId\":666,\"body\":{\"orderId\":1234567,\"refundId\":\"1234567-20180101160044458\",\"partnerOrderId\":\"partner-order-000001\",\"partnerDealId\":\"partner-deal-001\",\"refundQuantity\":1,\"unitPrice\":90.0,\"refundPrice\":90.0,\"refundFee\":0.0,\"refundTime\":\"2018-01-01 12:00:00\"}}"
}
```

#### 供应商返回参数

| 名称 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| code | Integer | 是 | 0 为正常，非 0 为异常 |
| message | String | 否 | 错误信息 or "success"，出错情况下，存放出错信息，用于问题定位。 |
| data | String | 否 | 在收到消息回调后若需主动返回数据，使用此字段 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ code | Integer (int32) | 是 | 交互码，必须回传；200 为退款成功，601 为已退款，602 为审批中，604 为产品本身不能退款，605 为产品不能线上退款，606 为申请张数超过可退款张数，607 为产品不在可退款时间内，608 为退款金额校验失败，699 为其它退款错误 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ describe | String | 是 | 交互描述，如错误信息等。失败时要返回真实的失败原因，不能用“其他”代替，必须回传 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ partnerId | Integer (int32) | 是 | 合作方 ID，必须回传 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ body | object<MtpOrderPayResponseBody> | 否 | 支付订单返回内容对象, 出票成功时必传 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ orderId | Integer (int64) | 是 | 美团订单 ID，必须回传 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ partnerOrderId | String | 是 | 合作方订单 ID |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ refundId | String | 是 | 美团退款流水 ID，商家必须存储和返回美团 refundid |
