## 消息场景说明
`meituan.trip.support.balance.query`(查询账户余额)

1. 我方主动调用此接口，获取合作方产品以及合作方账户的余额信息。
2. 我方用partnerId进行查询, 合作方需要准确告知商家账户信息, 如授信账户余额、预付款账户余额。
3. 我方用partnerId和partnerDealId进行查询，合作方需准确告知deal对应的商家账户信息, 如授信账户余额、预付款账户余额。

## 供应商返回参数

| 名称 | 类型 | 是否必须 | 描述 |
|------|------|----------|------|
| code | Integer | 是 | 0为正常，非0为异常 |
| message | String | 否 | 请求失败原因说明 |
| data | object | 否 | 扩展数据 |

### data结构

| 名称 | 类型 | 是否必须 | 描述 |
|------|------|----------|------|
| code | Integer (int32) | 是 | 交互码 取值请参考本接口的交互码类型映射表 |
| describe | String | 是 | 交互描述 字符串 |
| partnerId | Integer (int32) | 是 | 合作方ID，必须回传 |
| body | array | 是 | 账户余额信息 |

#### body对象

| 名称 | 类型 | 是否必须 | 描述 |
|------|------|----------|------|
| prepaidAccountBalance | Integer (int32) | 是 | 预付款账户余额。单位为分, 非预付款商家, 此值返回-1不要传0 |
| creditAccountBalance | Integer (int32) | 是 | 授信账户余额。单位为分, 无授信账户, 此值返回-1不要传0 |
| partnerDealId | String | 否 | 合作方产品ID。请求方法中包含合作方产品ID时,此项必填 |
