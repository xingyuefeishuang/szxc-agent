### 拉取价格日历

**更新时间：** 2026-03-09

**无需业务授权**

---

#### 消息场景说明

`meituan.trip.product.price.get` (拉取价格日历)
1.  我方主动调用此接口获取合作方产品的价格、库存等信息
2.  定时拉取，获取产品的价格信息动态
3.  支持单个查询

---

#### 公共参数

*   **HTTP URL：** `https://baseUrl`（供应商提供）（仅支持 https 协议且端口为 443）
*   **HTTP Param：** param 遵循 JSON 格式，ContentType 为 `application/x-www-form-urlencoded`, charset=utf-8
*   **HTTP 方法：** POST
*   **消息推送公共参数：** 消息推送公共参数
*   **消息响应公共规范：** 消息响应公共规范

---

#### 美团请求参数

| 参数 | 类型 | 是否必传 | 描述 |
| :--- | :--- | :--- | :--- |
| msgType | Integer | 是 | 接口类型 ID：6610017 |
| developerId | Integer | 是 | 开发者 ID，在美团开发者网页直连申请时生成 |
| opBizCode | String | 否 | 业务实体 id |
| businessId | Integer | 是 | 产线 ID，默认传 66 |
| msgId | String | 是 | 本条请求的唯一标识，可用于去重判断 |
| timestamp | Long | 是 | 消息推送时间戳 |
| message | String | 是 | json 格式的消息数据 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ partnerId | Integer (int32) | 是 | 第三方的 ID |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ body | object<MtpPriceRequestBody> | 是 | 请求价格日历内容 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ partnerDealId | String | 是 | 合作方产品 ID |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ startTime | String | 是 | 产品价格：开始时间 格式要求：yyyy-MM-dd |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ endTime | String | 是 | 产品价格：结束时间 格式要求：yyyy-MM-dd |

---

#### 请求示例

> **注意：** 回调消息统一通过 HTTPS 请求推送，Post 方法，ContentType 为 `application/x-www-form-urlencoded`。开发者需要按照规范实现 HTTPS 接口，以接收推送的消息。

**请求示例**
```plain text
{
    "msgType": "6610017",
    "opBizCode": "1711591987",
    "developerId": "123456",
    "businessId": "66",
    "sign": "4656285a4c2493e279d929b8b9f4e29310da8b3b",
    "msgId": "3702923312382104528",
    "timestamp": "1711592316026",
    "message": "{\"partnerId\":8851,\"body\":{\"partnerDealId\":\"1000000\",\"startTime\":\"2016-11-14\",\"endTime\":\"2017-02-12\"}}"
}
```

#### 供应商返回参数

结果数据应为 json 格式，放在 http body 中返回。

| 名称 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| code | Integer | 是 | 0 为正常，非 0 为异常 |
| message | String | 否 | 错误信息 or "success"，出错情况下，存放出错信息，用于问题定位。 |
| data | String | 否 | 在收到消息回调后若需主动返回数据，使用此字段 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ code | Integer (int32) | 是 | 交互码 取值请参考本接口的交互码类型映射表 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ describe | String | 是 | 交互描述 字符串 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ partnerId | Integer (int32) | 是 | 合作方 ID |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ body | array<MtpPriceResponseBody> | 是 | 价格集合 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ partnerDealId | String | 是 | 产品 id |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ date | String | 是 | 价格日历的日期 格式 yyyy-MM-dd |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ marketPrice | number (double) | 是 | 市场价。精确到 2 位小数; 单位: 元。 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ mtPrice | number (double) | 是 | 美团价。精确到 2 位小数; 单位: 元。 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ settlementPrice | number (double) | 是 | 结算价。精确到 2 位小数; 单位: 元。 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ stock | Integer (int32) | 是 | 日库存(指剩余可售卖库存) |
