### 拉取多层价格日历 V2

**更新时间：** 2026-03-09

**无需业务授权**

---

#### 消息场景说明

`meituan.trip.product.levelprice.get.v2` (拉取多层价格日历 V2)
1.  我方主动调用商家接口，从而获得价格库存信息
2.  该接口分为同步和异步两种情况，由请求参数中的 `asyncType` 标识，0 表示商家需同步返回，1 表示异步返回，商家后续通过 "多层价格日历变化通知 V2" 接口来推送多层价格日历

---

#### 公共参数

*   **HTTP URL：** `https://baseUrl`（供应商提供）（仅支持 https 协议且端口为 443）
*   **HTTP Param：** param 遵循 JSON 格式，ContentType 为 `application/x-www-form-urlencoded`, charset=utf-8
*   **HTTP 方法：** POST
*   **消息推送公共参数：** 消息推送公共参数
*   **消息响应公共规范：** 消息响应公共规范

---

#### 美团请求参数

| 参数 | 类型 | 是否必传 | 描述 |
| :--- | :--- | :--- | :--- |
| msgType | Integer | 是 | 接口类型 ID：6610021 |
| developerId | Integer | 是 | 开发者 ID，在美团开发者网页直连申请时生成 |
| opBizCode | String | 否 | 业务实体 id，参数与开发者 ID 一致 |
| businessId | Integer | 是 | 产线 ID，默认传 66 |
| msgId | String | 是 | 本条请求的唯一标识，可用于去重判断 |
| timestamp | Long | 是 | 消息推送时间戳（秒） |
| message | String | 是 | json 格式的消息数据 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ partnerId | int | 是 | 合作方 ID |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ body | object<MtpLevelPriceStockRequestBody> | 是 | 请求多层价格日历内容 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ partnerDealId | String | 是 | 合作方产品 ID |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ startTime | String | 是 | 开始时间，格式要求：yyyy-MM-dd |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ endTime | String | 是 | 结束时间 格式要求：yyyy-MM-dd |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ asyncType | int | 否 | 是否异步返回数据标识，0 表示同步,1 表示异步；默认为同步方式 |

---

#### 请求示例

> **注意：** 回调消息统一通过 HTTPS 请求推送，Post 方法，ContentType 为 `application/x-www-form-urlencoded`。开发者需要按照规范实现 HTTPS 接口，以接收推送的消息。

**请求示例**
```plain text
{
    "msgType": "6610021",
    "opBizCode": "1711591987",
    "developerId": "123456",
    "businessId": "66",
    "sign": "4656285a4c2493e279d929b8b9f4e29310da8b3b",
    "msgId": "3702923312382104528",
    "timestamp": "1711592316026",
    "message": "{\"partnerId\":1,\"body\":{\"partnerDealId\":\"example\",\"startTime\":\"example\",\"endTime\":\"example\",\"asyncType\":1}}"
}
```

#### 供应商返回参数

| 名称 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| code | Integer | 是 | 0 为正常，非 0 为异常 |
| message | String | 否 | 错误信息 or "success"，出错情况下，存放出错信息，用于问题定位。 |
| data | String | 否 | 在收到消息回调后若需主动返回数据，使用此字段 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ code | Integer | 是 | 交互码，必须回传；其中 200 为成功，999 为异步返回，交互码不能小于 100 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ describe | String | 是 | 交互描述，需要描述准确响应文案 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ partnerId | Integer | 是 | 合作方 ID，必须回传 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ partnerDealId | String | 是 | 商家产品 id |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ body | array<MtpTicketLevelPriceStockResponseBody> | 是 | 多层价格日历及场次信息集合 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ partnerPrimaryKey | String | 是 | sku 维度的唯一标识 id，建议生成方式：MD5(skuInfo.toJsonString+priceDate) |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ skuInfo | SkuInfo | 是 | sku 维度信息 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ startTime | String | 是 | 场次开始时间，要求格式：HH:mm，eg：15:30 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ endTime | String | 是 | 场次结束时间，要求格式：HH:mm，eg：20:59 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ levelInfoList | array<LevelInfo> | 是 | 场次信息 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ levelNo | Integer | 是 | 层级序号，最多支持 3 层：1-第一层；2-第二层；3-第三层 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ levelName | String | 是 | 层级名称信息，如：”上午场“、”贵宾席“ |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ priceDate | String | 是 | 价格日历的日期, 格式 yyyy-MM-dd eg：2022-10-01 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ marketPrice | Double | 是 | 市场价，单位：元，保留两位小数 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ mtPrice | Double | 是 | 美团售卖价，单位：元，保留两位小数 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ settlementPrice | Double | 是 | 结算价，单位：元，保留两位小数 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ stock | Integer | 是 | 日库存(指剩余可售卖库存) |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└ attr | String | 否 | 扩展数据，sku 信息，如：`"{\"playtime\":\"09:00-10:00\",\"ticketcode\":\"贵宾席\",\"stock\":100}"` |
