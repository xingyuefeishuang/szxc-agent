# 景区核销

**更新时间**: 2025-08-12 15:35:02

## 接口说明

*   核销凭证，必传字段 `order_id`，现状默认整单核销。
*   如果接口失败请重试，控制重试策略不要太频繁，核销调用失败会有兜底核销逻辑。
*   不支持提前核销（特殊场景提前核销，可向对接 BD 提需）。
*   如商品为日历票多门店，必须在核销时回传核销门店 `poi`。
*   若抖音返回失败，则服务商需要重试接口。

**整单核销示例**:
```json
{ 
  "verify_token": "ADVSADFA", 
  "order_id": "8000004478714899970" 
}
```

**部分核销示例**:
```json
{ 
  "verify_token": "1681885242", 
  "order_id": "1003051470049948030", 
  "vouchers": [ 
    {
      "projects": {
        "project_id": "a1"
      }
    }
  ] 
}
```

| 名称 | 描述 |

## 请求参数

### 请求头

| Key | 字段类型 | 示例 | 描述 |
| :--- | :--- | :--- | :--- |
| `access-token` | String (必填) | `clt.943da...` | 调用 `https://open.douyin.com/oauth/client_token/` 生成的 token |

### 请求体 (Body)

| Key | 字段类型 | 示例 | 描述 |
| :--- | :--- | :--- | :--- |
| `order_id` | String (必填) | - | 抖音侧的订单号 |

| `poi_id` | String | - | 抖音 POI_ID（即门店 id）。如商品为日历票多门店，核销门店 poi 为必传 |
| `verify_time` | Int64 | - | 自定义核销时间，时间戳。秒级时间戳 |
| `vouchers` | List | - | 本次核销的券码（部分退需要） |
| `vouchers[].entrance` | Struct | - | - |
| `vouchers[].entrance.project_id` | String (必填) | - | 入园项目的唯一标识，与 projects 中不能重复（核销时需要） |
| `vouchers[].entrance.certificate_nos` | Array | - | 凭证列表 |

| `vouchers[].entrance.gmcode_imgs` | Array | - | - |
| `vouchers[].entrance.id_cards` | Array | - | - |

| `vouchers[].entrance.qrcodes` | Array | - | - |
| `vouchers[].entrance.urls` | Array | - | - |
| `vouchers[].projects` | List | - | - |
| `vouchers[].projects[].project_id` | String (必填) | - | - |
| `vouchers[].projects[].certificate_nos` | Array | - | - |
| `vouchers[].projects[].credentials` | List | - | - |
| `vouchers[].projects[].credentials[].credential_no` | String | - | - |
| `vouchers[].projects[].credentials[].credential_type` | Enum | - | 凭证类型同上 |
| `vouchers[].projects[].gmcode_imgs` | Array | - | - |

| `extra` | Struct | - | - |
| `extra.description` | String (必填) | - | 错误码描述 |
| `extra.error_code` | Int32 (必填) | `0` | 错误码 |
