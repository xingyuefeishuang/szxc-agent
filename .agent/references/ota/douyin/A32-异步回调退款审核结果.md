# 回调审核退款

## 接口说明

本接口用于服务商在收到抖音侧的**申请退款 SPI** 请求后，**异步回调**最终的审核结果。

*   **触发场景**：当服务商无法同步返回审核结果（即申请退款 SPI 返回了“等待审核”或系统繁忙需重试），处理完成后需调用此接口通知抖音最终决定。
*   **审核逻辑**：
    *   当审核结果为**接受** (`Accept`) 时：
        *   若传入的 `refund_fee_amount` (手续费) **小于等于** 平台计算的手续费，以**平台计算罚金**为准。
        *   若传入的手续费 **大于** 平台计算手续费，视为**拒绝**。
    *   **特殊场景**：在**取消预约**场景中，这里的退款金额可能为零。
*   **重试机制**：若抖音返回失败（`error_code` 非 0），服务商需要**持续重试**该接口直至成功。

## 请求参数

### 请求头 (Headers)

| Key | 字段类型 | 示例 | 描述 |
| :--- | :--- | :--- | :--- |
| `access-token` | String (必填) | `clt.943da...` | 调用 `https://open.douyin.com/oauth/client_token/` 生成的 token |
| `content-type` | String (必填) | `application/json` | 固定值 |

### 请求体 (Body)

| Key | 字段类型 | 示例 | 描述 |
| :--- | :--- | :--- | :--- |
| `order_id` | String (必填) | `10001` | 抖音侧订单 ID |
| `biz_uniq_key` | String (必填) | `123456` | **业务唯一键** (对应申请退款 SPI 中的 `biz_uniq_key`，用于幂等) |
| `audit_result` | Enum (必填) | `1` | **退款审核结果**：`1`: 接受 (`Accept`)`2`: 拒绝 (`Reject`) |
| `refund_fee_amount` | Int64 | `2340` | **退款手续费金额** (单位：分)**注意**：仅当审核结果为**接受**时必须传递。 |
| `vouchers` | List | - | **凭证列表**(主要用于**补码**场景，如发码失败导致的退款需补发新码) |
| `verify_time` | Int64 | - | 自定义核销时间 (子履约单)，单位秒 |

#### `vouchers` 列表项结构

若需补发凭证，需构造此列表。包含入园项目和自定义项目。

| Key | 字段类型 | 描述 |
| :--- | :--- | :--- |
| `entrance` | Struct | **景区的入园项目**(若入园凭证是单独凭证，用此字段；否则可用 `projects` 传入) |
| `projects` | List | **可自定义的项目** (例：索道 A、索道 B) |

##### `entrance` / `projects` 子项字段详解

| Key | 字段类型 | 描述 |
| :--- | :--- | :--- |
| `project_id` | String (必填) | 项目唯一标识，同一个订单下不能重复（核销时需要） |
| `name` | String | 项目名称 (自定义项目必填) |
| `certificate_nos` | Array | 券号凭证，在 C 端展示为数字串，最多限 100 个 |
| `credentials` | List | 证件列表 |
| `qrcodes` | Array | 二维码凭证，展示为二维码，每个字符串长度不能超过 512 |
| `urls` | Array | 链接凭证，在 C 端直接展示此链接 (需加白使用) |
| `id_cards` | Array | 身份证号码，最多限 100 个 |
| `gmcode_imgs` | Array | 图片码 (如有) |

###### `credentials` (证件列表) 子项字段

| Key | 字段类型 | 描述 |
| :--- | :--- | :--- |
| `credential_no` | String (必填) | 凭证号 |
| `credential_type` | Enum (必填) | **凭证类型**：`1`: 身份证 (`ID_CARD`)`2`: 港澳通行证 (`HM_PERMIT`)`3`: 台湾通行证 (`TW_PERMIT`)`4`: 回乡证 (`REENTRY_PERMIT`)`5`: 台胞证 (`TW_MAINLAND_TRAVEL_PERMIT`)`6`: 护照 (`PASSPORT`) |

## 请求示例 (Java)

```java
import com.aliyun.tea.TeaException;
import com.douyin.openapi.client.Client;
import com.douyin.openapi.client.models.*;
import com.douyin.openapi.credential.models.Config;

public class Main {
    public static void main(String[] args) {
        try {
            // 配置 ClientKey 和 ClientSecret
            Config config = new Config().setClientKey("tt******").setClientSecret("cbs***"); 
            Client client = new Client(config);
            
            /* 
               构建请求参数：
               1. token: 可自行维护或调用 SDK 获取
               2. header: SDK 默认填充 content-type
            */
            TripRefundAuditRequest sdkRequest = new TripRefundAuditRequest();
            sdkRequest.setAccessToken("k23cd*****");
            sdkRequest.setOrderId("nkxCoKh3mc");          // 抖音订单号
            sdkRequest.setBizUniqKey("FNlrKc7p1Q");        // 业务唯一键
            sdkRequest.setAuditResult(1);                  // 1=接受, 2=拒绝
            sdkRequest.setRefundFeeAmount(1L);             // 手续费 (接受时必传)
            
            // 若有补码需求，可在此处设置 vouchers 字段
            // sdkRequest.setVouchers(...);

            // 发送请求
            TripRefundAuditResponse sdkResponse = client.TripRefundAudit(sdkRequest);
            
        } catch (TeaException e) {
            System.out.println("TeaException: " + e.getMessage());
        } catch (Exception e) {
            System.out.println("Exception: " + e.getMessage());
        }
    }
}
```
