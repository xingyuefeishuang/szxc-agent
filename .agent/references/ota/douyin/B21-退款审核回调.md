# 景区回调审核退款

**更新时间**: 2025-08-12 15:34:01

## 接口说明

抖音侧请求第三方申请退款后，第三方异步回调审核结果。当退款审核结果为接受，若传入手续费小于等于平台计算手续费，以平台计算罚金为准，否则视为拒绝。

> **注意**：取消预约场景这里的退款金额为零。
>
> 若抖音返回失败，则需要重试接口。

## 基本信息

| 名称 | 描述 |
| :--- | :--- |
| **HTTP URL** | `https://open.douyin.com/goodlife/v1/trip/refund/audit/` |
| **HTTP Method** | `POST` |
| **Scope** | `life.capacity.trip_order_after_sale_refund_audit` |
| **权限要求** | 景区回调审核退款 |

### 请求体 (Body)

| Key | 字段类型 | 示例 | 描述 |
| :--- | :--- | :--- | :--- |
| `audit_result` | Enum (必填) | `1` | 退款审核结果：`1`: 接受 (Accept)`2`: 拒绝 (Reject) |
| `biz_uniq_key` | String (必填) | `123456` | 业务唯一键 |

| `refund_fee_amount` | Int64 | `2340` | **通过审核时必须传**，退款手续费金额（单位：分） |
| `vouchers` | List | - | 凭证列表（补码需要） |
| `vouchers[].entrance` | Struct | - | 景区的入园项目。*(若入园凭证是单独的凭证，则使用此字段；若不是单独的凭证，可使用自定义项目的字段传入)* |
| `vouchers[].entrance.project_id` | String (必填) | - | 入园项目的唯一标识，同一个订单下不能重复（核销时需要） |
| `vouchers[].entrance.certificate_nos` | Array | - | 券号凭证，最多限 100 个 |
| `vouchers[].entrance.credentials` | List | - | 证件列表 |

| `vouchers[].entrance.credentials[].credential_type` | Enum | - | 凭证类型：`1`: 身份证 (ID_CARD)`2`: 港澳通行证 (HM_PERMIT)`3`: 台湾通行证 (TW_PERMIT)`4`: 回乡证 (REENTRY_PERMIT)`5`: 台胞证 (TW_MAINLAND_TRAVEL_PERMIT)`6`: 护照 (PASSPORT) |
| `vouchers[].entrance.id_cards` | Array | - | 身份证号码，最多限 100 个 |
| `vouchers[].entrance.name` | String | - | 项目名称 |
| `vouchers[].entrance.qrcodes` | Array | - | 二维码凭证内容，会生成二维码展示，每个字符串长度不能超过 512 |
| `vouchers[].entrance.urls` | Array | - | 链接凭证，在 C 端直接展示此链接（需加白使用） |
| `vouchers[].projects` | List | - | 可自定义的项目（例：景区项目索道 A、索道 B 等） |
| `vouchers[].projects[].project_id` | String (必填) | - | 入园项目的唯一标识，同一个订单下不能重复（核销时需要） |
| `vouchers[].projects[].certificate_nos` | Array | - | 券号凭证，在 C 端展示为数字串，最多限 100 个 |
| `vouchers[].projects[].credentials` | List | - | 证件列表（结构同 `entrance.credentials`） |
| `vouchers[].projects[].name` | String | - | 自定义的项目名称（例：景区 XXXX 索道） |
| `vouchers[].projects[].qrcodes` | Array | - | 二维码凭证，展示为二维码，每个 URL 长度不能超过 512 |
| `vouchers[].projects[].urls` | Array | - | 链接凭证，在 C 端直接展示此链接（需加白使用） |
| `verify_time` | Int64 | - | 自定义核销时间 (子履约单)，单位秒 |
