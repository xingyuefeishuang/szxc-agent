# 履约验券

> **收藏**: [我的收藏](#)

## 接口说明

本接口用于**核销凭证**。

*   **必传字段**: `order_id`。
*   **核销模式**: 现状默认**整单核销**。若需部分核销，需传递 `vouchers` 参数。
*   **重试策略**: 如果接口调用失败，服务商需要重试接口。请控制重试频率，不要过于频繁。核销调用失败时，抖音侧会有兜底核销逻辑。
*   **提前核销**: 默认**不支持**提前核销。如有特殊场景需求，需向对接的 BD 提需。
*   **多门店场景**: 如商品为**日历票多门店**，必须在核销时回传核销门店 `poi_id`。

### 请求体示例

**整单核销**:
```json
{
  "verify_token": "ADVSADFA",
  "order_id": "8000004478714899970"
}
```

**部分核销**:
```json
{
  "verify_token": "1681885242",
  "order_id": "1003051470049948030",
  "vouchers": [
    {
      "projects": [
        {
          "project_id": "a1"
        }
      ]
    }
  ]
}
```

### 请求头 (Headers)

| Key | 字段类型 | 示例 | 描述 |

| `content-type` | String (必填) | `application/json` | 固定值 |

### 请求体 (Body)

| Key | 字段类型 | 描述 |
| :--- | :--- | :--- |
| `order_id` | String (必填) | 抖音侧的订单号 |
| `verify_token` | String (必填) | 一次验券的标识 (用于短时间内的幂等) |
| `poi_id` | String | 抖音 POI_ID（即门店 id）。**注意**: 如商品为日历票多门店，此字段为**必传**。 |
| `verify_time` | Int64 | 自定义核销时间，秒级时间戳 |
| `vouchers` | List | 本次核销的券码列表（**部分核销/部分退需要**） |

#### `vouchers` 列表项结构

| Key | 字段类型 | 描述 |
| :--- | :--- | :--- |
| `entrance` | Struct | 入园项目信息 |
| `projects` | List | 自定义项目列表 |
| `verify_time` | Int64 | 自定义核销时间 (子履约单)，单位秒 |

##### `entrance` (入园项目) 字段

| Key | 字段类型 | 描述 |
| :--- | :--- | :--- |
| `project_id` | String (必填) | 入园项目的唯一标识，与 `projects` 中不能重复（核销时需要） |
| `certificate_nos` | Array | 凭证列表 |
| `credentials` | List | 身份凭证列表 |
| `gmcode_imgs` | Array | - |
| `id_cards` | Array | - |
| `name` | String | - |
| `qrcodes` | Array | - |
| `urls` | Array | - |

###### `credentials` (身份凭证) 列表项字段

| Key | 字段类型 | 枚举值/描述 |
| :--- | :--- | :--- |
| `credential_no` | String | 证件号 |
| `credential_type` | Enum | 证件类型：`1`: 身份证 (`ID_CARD`)`2`: 港澳通行证 (`HM_PERMIT`)`3`: 台湾通行证 (`TW_PERMIT`)`4`: 回乡证 (`REENTRY_PERMIT`)`5`: 台胞证 (`TW_MAINLAND_TRAVEL_PERMIT`)`6`: 护照 (`PASSPORT`) |

##### `projects` (自定义项目) 列表项字段

| Key | 字段类型 | 描述 |
| :--- | :--- | :--- |
| `project_id` | String (必填) | 项目唯一标识 |
| `certificate_nos` | Array | 凭证列表 |
| `credentials` | List | 身份凭证列表 (结构同上) |
| `gmcode_imgs` | Array | - |

import com.aliyun.tea.TeaException;
import com.douyin.openapi.client.Client;
import com.douyin.openapi.client.models.*;
import com.douyin.openapi.credential.models.Config;

public class Main {
    public static void main(String[] args) {
        try {
            // 配置 ClientKey 和 ClientSecret
            Config config = new Config().setClientKey("tt******").setClientSecret("cbs***"); 
            Client client = new Client(config);
            
            /* 
               构建请求参数：
               1. token: 可自行维护或调用 SDK 获取 (注意互刷问题)
               2. header: SDK 默认填充 content-type，其他请求头按需添加

            sdkRequest.setAccessToken("k23cd*****");
            sdkRequest.setOrderId("ynbE02XxO0");       // 抖音订单号

        } catch (TeaException e) {
            System.out.println("TeaException: " + e.getMessage());
        } catch (Exception e) {
            System.out.println("Exception: " + e.getMessage());
        }
    }
}
