# 确认订单

## 接口说明

抖音侧通知第三方支付成功后，第三方需要在十分钟内通知抖音侧接单结果。

- 非必要不接入，建议服务商适配同步接单，同步接单可以不对接该接口
- 若抖音返回失败，则需要重试接口

---

## 基本信息

| 名称 | 描述 |
|------|------|
| **HTTP URL** | `https://open.douyin.com/goodlife/v1/trip/order/confirm/` |
| **HTTP Method** | POST |
| **Scope** | `life.capacity.trip_order_confirm` |
| **权限要求** | 景区确认订单 |

---

## 请求参数

### 请求头

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| access-token | String | 是 | 调用 `https://open.douyin.com/oauth/client_token/` 生成的 token |
| content-type | String | 是 | 固定值 `application/json` |

### Body

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| confirm_result | Enum | 是 | 确认订单结果：1 接受，2 拒绝 |
| order_id | String | 是 | 抖音侧订单号 |

#### confirm_result 枚举值

| 值 | 描述 |
|----|------|
| 1 | 接受 |
| 2 | 拒绝 |

                .setClientSecret("cbs***"); // 改成自己的 app_id 跟 secret
