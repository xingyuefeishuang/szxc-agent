# 景区退款审核

**更新时间**: 2026-03-11 19:56:12  
**收藏**: 我的收藏

## 接口介绍

*   **POST**
*   抖音侧请求第三方申请退款，第三方同步或异步回调审核结果。
*   未预约的预售单退款不送审，直接通过【退款结果通知】进行通知。

## 基本信息

| 项目 | 描述 |
| :--- | :--- |
| **Scope** | `life.capacity.trip_order_after_sale_refund_apply` |
| **权限要求** | 景区行业解决方案 - 景区退款审核 |
| **回调场景** | 景区申请退款 |

## 请求参数

| 参数名称 | 参数类型 | 是否必填 | 是否加密 | 参数描述 |
| :--- | :--- | :---: | :---: | :--- |
| `order_id` | string | 是 | 否 | 抖音订单 ID，如果是预约场景，对应预约订单 ID |
| `out_order_id` | string | 否 | 否 | 商户订单 id |
| `biz_uniq_key` | string | 是 | 否 | 业务唯一键，用于审核回调（可作幂等键） |
| `after_sale_type` | int32 | 是 | 否 | 售后类型：1-退款；2-取消预约 |
| `apply_source` | int64 | 否 | 否 | 申请退款类型301 发码失败 (发码失败拒绝退款需要返回补码信息)302 过期自动退 |
| `apply_refund_time_unix` | int64 | 是 | 否 | 申请时间戳，秒 |
| `vouchers` | list | 否 | 否 | 凭证列表（只有部分退/取消时，且非发码失败，才会给该字段，标识涉及取消或退款的凭证） |
| `-projects` | list | 否 | 否 | 可自定义的项目（例：景区项目索道 A、索道 B 等） |
| `--project_id` | string | 是 | 否 | 项目唯一标识，不能重复（核销时需要） |

## 响应字段

| 参数名称 | 参数类型 | 是否必填 | 是否加密 | 参数描述 |
| :--- | :--- | :---: | :---: | :--- |
| `data` | struct | 是 | 否 | - |
| `- error_code` | int | 是 | 否 | 错误码，合法范围为 [0,999999] |
| `- description` | string | 是 | 否 | 错误信息 |
| `- audit_refund_result` | int | 是 | 否 | 审核退款结果，当处理成功时必须返回，1 接受，2 拒绝，3 等待审核 |
| `- refund_fee_amount` | int | 否 | 否 | 退款手续费金额，退款场景，审核退款结果为接受时必须返回，分；抖音侧如果校验此金额大于抖音侧计算的手续费金额则拒绝退款（团购场景需返回默认值 0） |
| `-vouchers` | list | 否 | 否 | 凭证列表（补码需要） |
| `--projects` | list | 否 | 否 | 可自定义的项目（例：景区项目索道 A、索道 B 等） |
| `---project_id` | string | 是 | 否 | 入园项目的唯一标识，同一个订单下不能重复（核销时需要） |
| `---name` | string | 是 | 否 | 自定义的项目名称（例：景区 XXXX 索道） |
| `---id_cards` | list | 否 | 是 | 身份证号码，最多限 100 个 |
| `---qrcodes` | list | 否 | 否 | 二维码凭证，展示为二维码，每个 URL 长度不能超过 512。 |
| `---urls` | list | 否 | 否 | url 凭证 |
| `---certificate_nos` | list | 否 | 否 | 券号凭证，在 C 端展示为数字串，最多限 100 个 |
| `---credentials` | list | 否 | 否 | 证件列表 |
| `----credential_type` | int | 是 | 否 | 凭证类型1:身份证 2:港澳通行证 3:台湾通行证 4:回乡证 5:台胞证 6:护照 |
| `----credential_no` | string | 是 | 是 | 凭证号 |
