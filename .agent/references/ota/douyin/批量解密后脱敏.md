# 开放平台解密脱敏接口

**更新时间**: 2025-12-10 15:04:56

## 接口说明

本接口用于批量解密密文数据并直接进行**脱敏处理**。
*   **适用场景**: 适用于只需要展示部分信息（如手机号中间四位掩码、姓名只显示姓氏）的场景，无需获取完整明文。
*   **优势**: 相比普通解密接口，此接口**不消耗**每日解密配额，且能降低敏感数据泄露风险。

### 使用限制
*   **SLA**: 支持的最大 QPS 为 **100**。

## 请求示例

```bash

--header 'access-token: 0801121846735352506a356a6' \
--header 'content-type: application/json' \
--header 'Rpc-Transit-Life-Account: niz7jzCf0P' \
--data '{

  "cipher_texts": ["PzwTHJeAJn"]
```
