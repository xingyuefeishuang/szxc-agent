# 开放平台加密接口

**更新时间**: 2025-12-10 15:05:05

## 接口说明

本接口用于批量将明文数据加密为密文。
*   **适用场景**: 当您需要向平台方安全地回传敏感数据时（例如：交易回调、信息上报、用户信息更新等），必须使用此接口对明文进行加密。
*   **权限要求**: 需具备“开放平台加密接口”权限。

### 使用限制
*   **SLA**: 支持的最大 QPS 为 **100**。

## 基本信息

| 名称 | 描述 |
| :--- | :--- |
| **HTTP URL** | `https://open.douyin.com/goodlife/v1/open/common_biz/crypto/encrypt/batch/` |
| **HTTP Method** | `POST` |
| **Scope** | `life.capacity.open.common_biz.crypto.encrypt.batch` |

## 请求参数

### 请求头 (Headers)

| Key | 字段类型 | 示例 | 描述 |
| :--- | :--- | :--- | :--- |
| `access-token` | String | `clt.943da17996fb5cebfbc70c044c3fc25a57T54DcjT6HNKGqnUdxzy1KcxFnZ` | **必填**。调用 `client_token` 接口生成的凭证。 |
| `content-type` | String | - | **必填**。固定值 `"application/json"`。 |
| `Rpc-Transit-Life-Account` | String | - | **可选**。来客商户根账户 ID。 |

### 请求体 (Body)

| Key | 字段类型 | 示例 | 描述 |
| :--- | :--- | :--- | :--- |
| `account_id` | String | - | **必填**。账户 ID。 |
| `plain_texts` | List | - | **必填**。待加密的明文列表。 |
| └─ `text` | String | `wc7l1y19v9` | **必填**。原始明文内容。 |
| └─ `text_type` | Enum | `1` | **必填**。数据类型枚举值：`0`: UNKNOWN (未知)`1`: UID (用户ID)`2`: Phone (手机号)`3`: Name (姓名)`4`: IDCard (身份证号)`5`: Other (其他)`6`: Address (地址) |

## 请求示例

```bash
curl --location --request POST 'https://open.douyin.com/goodlife/v1/open/common_biz/crypto/encrypt/batch/' \
--header 'content-type: application/json' \
--header 'access-token: 0801121846735352506a356a6' \
--header 'Rpc-Transit-Life-Account: 5sGKeKUn8F' \
--data '{
  "account_id": "pVjkxbcZwT",
  "plain_texts": [
    {
      "text": "wc7l1y19v9",
      "text_type": 1
    }
  ]
}'
```

## 响应参数

### 响应体 (Body)

| Key | 字段类型 | 示例 | 描述 |
| :--- | :--- | :--- | :--- |
| `data` | Struct | - | **必填**。业务数据主体。 |
| └─ `description` | String | `"access_token 无效"` | **必填**。错误描述（成功时通常为空）。 |
| └─ `error_code` | Int32 | `28001003` | **必填**。整体错误码（0 表示成功）。 |
| └─ `failure_count` | Int64 | - | 加密失败的条目数量。 |
| └─ `results` | List | - | 加密结果列表。 |
| &nbsp;&nbsp;└─ `[Array Item]` | Struct | - | **必填**。单个加密结果项。 |
| &nbsp;&nbsp;&nbsp;&nbsp;├─ `cipher_text` | String | - | 加密后的密文（若失败则为空）。 |
| &nbsp;&nbsp;&nbsp;&nbsp;├─ `plain_text` | String | - | 原始明文（回显，方便核对）。 |
| &nbsp;&nbsp;&nbsp;&nbsp;├─ `is_success` | Bool | - | 该条数据是否加密成功。 |
| &nbsp;&nbsp;&nbsp;&nbsp;├─ `error_code` | Int32 | - | 该条数据的错误码。 |
| &nbsp;&nbsp;&nbsp;&nbsp;└─ `error_msg` | String | - | 该条数据的错误信息。 |
| `extra` | Struct | - | **必填**。扩展信息。 |
| └─ `logid` | String | - | **必填**。抖音日志 ID，可用于排查问题。 |
| └─ `now` | Int64 | - | **必填**。调用时间戳，单位为秒 (s)。 |
| └─ `error_code` | Int32 | - | **必填**。扩展信息中的错误码。 |
| └─ `description` | String | - | **必填**。扩展信息中的描述。 |
| └─ `sub_error_code` | Int32 | - | **必填**。子错误码。 |
| └─ `sub_description` | String | - | **必填**。子描述。 |
